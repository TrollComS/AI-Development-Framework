# AI Development Framework (ADF)

O ADF organiza o trabalho conjunto entre pessoas e IAs no ciclo de desenvolvimento. Contexto, decisões, requisitos e procedimentos tornam-se artefatos versionados, navegáveis e verificáveis.

## Princípios

- documentação como parte do produto;
- contexto explícito antes da execução;
- rastreabilidade entre regra, decisão, feature, código e teste;
- entregas pequenas com critérios objetivos;
- independência de tecnologia e fornecedor de IA;
- revisão humana para decisões e ações de risco.
