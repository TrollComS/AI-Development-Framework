# Feature: Exportar relatório

**Estado:** Exemplo

## Problema

Pessoas autorizadas precisam analisar dados fora do sistema.

## Resultado esperado

Um relatório filtrado pode ser exportado em formato aberto.

## Fora do escopo

Agendamento e envio automático.

## Critérios de aceite

- [ ] Dado um filtro válido, quando a exportação terminar, então o arquivo contém somente os registros filtrados.
- [ ] Dado que não há registros, então o arquivo mantém os cabeçalhos.

Este sample é ilustrativo e não define regra de nenhum projeto consumidor.
