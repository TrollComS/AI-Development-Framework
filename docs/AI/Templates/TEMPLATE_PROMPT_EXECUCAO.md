# Prompt: executar mudança

Atue como IA Executora. Comece em `docs/INDICE_DOCUMENTACAO.md`, leia a feature, a skill indicada e somente o contexto relacionado. Confirme escopo e critérios; preserve mudanças não relacionadas; implemente o menor conjunto coerente; revise o diff; execute build e testes aplicáveis; avalie e atualize a documentação afetada. Encerre com arquivos alterados, validações, documentos avaliados/alterados e pendências.
