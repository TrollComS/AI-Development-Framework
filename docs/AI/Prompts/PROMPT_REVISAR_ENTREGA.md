# Prompt: revisar entrega

Revise a mudança contra a feature e os artefatos referenciados. Priorize defeitos, regressões, segurança, compatibilidade, ausência de testes e divergência documental. Para cada achado, indique gravidade, evidência, localização e correção esperada. Se não houver achados, declare as verificações realizadas e os riscos residuais.
