# Prompt: iniciar feature

Analise a feature indicada seguindo o ADF. Leia primeiro o índice, a configuração de IAs do projeto e somente o contexto relacionado. Separe fatos, inferências e lacunas. Verifique se os critérios de aceite são objetivos. Defina a rota de execução entre IA Pensante, IA Dev Principal e OpenCode conforme `docs/AI/Core/ROTEAMENTO_IAS.md`. Antes de implementar, apresente um plano incremental com testes, documentação e reversão. Preserve mudanças não relacionadas e peça autorização para ações irreversíveis.
