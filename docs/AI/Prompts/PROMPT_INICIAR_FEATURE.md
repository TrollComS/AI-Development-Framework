# Prompt: iniciar feature

Analise a feature indicada seguindo o ADF. Leia primeiro o índice e somente o contexto relacionado. Separe fatos, inferências e lacunas. Verifique se os critérios de aceite são objetivos. Antes de implementar, apresente um plano incremental com testes, documentação e reversão. Preserve mudanças não relacionadas e peça autorização para ações irreversíveis.
