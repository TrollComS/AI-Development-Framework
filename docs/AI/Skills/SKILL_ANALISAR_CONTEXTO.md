# Skill: Analisar contexto

**Entrada:** tarefa e repositório. **Saída:** fontes relevantes, lacunas e restrições.

1. Confirmar objetivo e limite.
2. Ler índice, visão e glossário.
3. Localizar regras, ADRs, padrões e features relacionados.
4. Inspecionar estado atual do código.
5. Separar fatos, inferências e dúvidas.
6. Carregar somente contexto necessário.

Não implementar durante esta skill.
