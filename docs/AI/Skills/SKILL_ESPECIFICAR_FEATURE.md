# Skill: Especificar feature

**Entrada:** necessidade validada. **Saída:** feature pronta.

1. Definir problema e resultado observável.
2. Delimitar dentro e fora do escopo.
3. Referenciar regras e decisões.
4. Registrar requisitos, riscos e dependências.
5. Escrever critérios verificáveis.
6. Resolver ambiguidades que mudem materialmente a solução.

Use o template de feature.
