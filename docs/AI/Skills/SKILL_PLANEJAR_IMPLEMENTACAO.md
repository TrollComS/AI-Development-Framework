# Skill: Planejar implementação

**Entrada:** feature pronta. **Saída:** plano incremental.

1. Mapear componentes afetados.
2. Definir passos pequenos e dependências.
3. Planejar migração e compatibilidade.
4. Associar testes a cada critério.
5. Prever observabilidade, documentação e reversão.
6. Sinalizar decisões ainda necessárias.

O plano descreve resultados, não atividade genérica.
