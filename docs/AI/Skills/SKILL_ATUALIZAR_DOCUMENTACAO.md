# Skill: Atualizar documentação

**Entrada:** mudança validada. **Saída:** documentação coerente.

1. Identificar comportamento e decisões alterados.
2. Atualizar feature, regra, ADR ou padrão na fonte.
3. Atualizar índice e mapas se a estrutura mudou.
4. Corrigir links e remover instruções obsoletas.
5. Registrar versão quando o contrato público mudou.
6. Conferir que documentação e sistema descrevem o mesmo estado.
