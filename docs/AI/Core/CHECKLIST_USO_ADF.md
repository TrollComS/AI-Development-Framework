# Checklist de uso do ADF

- [ ] O índice principal foi consultado.
- [ ] Objetivo, escopo e conclusão estão claros.
- [ ] Papel e skill adequados foram escolhidos.
- [ ] Somente o contexto relacionado foi carregado.
- [ ] Critérios de aceite são verificáveis.
- [ ] Mudanças locais não relacionadas foram preservadas.
- [ ] Segurança, compatibilidade e regressões foram avaliadas.
- [ ] Build e testes aplicáveis foram executados ou o impedimento foi registrado.
- [ ] Documentação afetada foi avaliada e atualizada quando necessário.
- [ ] Arquivos alterados, validações e pendências constam da entrega.
